# Caddy Webserver

This is an example project demonstrating an understanding of using Caddy to
swiftly spin up a basic webserver that serves (bundled) static sites.

# Usage
```sh
gh repo clone cyrus01337/caddy-webserver && \
    cd caddy-webserver
```
To start the application, run the following:
```sh
docker compose up -d
```
Then visit [localhost](https://localhost/).
